pySEAFOM

Performance analysis and testing tools for Distributed Acoustic Sensing (DAS) interrogators.

GitHub PyPI README

Modules

  • Self-noise – pySEAFOM.self_noise
  • Dynamic range – pySEAFOM.dynamic_range
  • Fidelity (THD) – pySEAFOM.fidelity
  • Crosstalk – pySEAFOM.crosstalk

Quick start

pip install pySEAFOM

For examples and workflows, see the notebooks in testing_notebooks and the guides in workflows.
